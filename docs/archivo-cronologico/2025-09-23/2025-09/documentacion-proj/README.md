# Documentación del Proyecto

Este proyecto tiene como objetivo desarrollar un sistema de gestión de preinscripciones, incluyendo funcionalidades para la autenticación de usuarios, gestión de pagos, y un panel administrativo para la supervisión de procesos.

## Documentos Clave

- **Esquema de Sprints.md**: Detalla la planificación de sprints, incluyendo tareas, descripciones, criterios de aceptación y sprints sugeridos.
- **DRS_Final.md**: Consolidación de la versión final del Documento de Requerimientos.
- **Metodologia_Sprints.md**: Describe la metodología para la gestión de sprints, incluyendo roles y calendario inicial.
- **ERD_Modelo_Datos.md**: Presenta el Diagrama de Entidad-Relación (ERD) del modelo de datos.
- **Plan_Migracion.md**: Detalla el plan de migración de datos y el mapeo de columnas a campos de base de datos.
- **Prototipo_Landing.md**: Contiene el prototipo de alta fidelidad de la página de inicio.
- **Prototipo_Wizard_Preinscripcion.md**: Incluye los mockups del asistente de preinscripción.
- **Configuracion_Repos_CICD.md**: Describe la configuración de repositorios y procesos de CI/CD.
- **Autenticacion_Usuarios_API.md**: Detalla los endpoints de la API para la autenticación de usuarios.
- **Esquema_Roles_RBAC.md**: Describe el esquema de control de acceso basado en roles (RBAC).
- **API_CRUD_Preinscripciones.md**: Detalla la API para gestionar datos de preinscripción.
- **Subida_Archivos_Segura.md**: Describe el proceso de subida segura de archivos.
- **Importador_Pagos_Grupales.md**: Detalla el importador de pagos grupales.
- **Panel_Finanzas.md**: Describe el panel financiero para gestionar pagos individuales.
- **Regla_Confirmacion_Pago.md**: Describe la regla configurable para confirmar pagos.
- **Panel_Admin_Habilitacion.md**: Detalla el panel administrativo para habilitar usuarios.
- **Dashboards_KPIs.md**: Describe los dashboards iniciales para el seguimiento de indicadores clave.
- **Panel_Validadores.md**: Detalla el panel para la gestión de aprobaciones.
- **Panel_Acreditacion_Manual.md**: Describe el panel de acreditación manual.
- **Comunicacion_Envios_Masivos.md**: Detalla el proceso para registrar comunicaciones masivas.
- **Pruebas_Automatizadas.md**: Describe la estrategia de pruebas automatizadas.
- **Despliegue_Piloto_Formacion.md**: Detalla el plan de despliegue piloto y sesiones de capacitación.

## Instrucciones de Configuración

1. Clonar el repositorio.
2. Instalar las dependencias necesarias.
3. Configurar las variables de entorno según el archivo de configuración.
4. Ejecutar el servidor de desarrollo.
