# Panel de Finanzas

## Descripción
Este documento describe el panel de finanzas, que permite gestionar los pagos individuales de los participantes. Incluye funcionalidades para buscar registros de pago, registrar nuevos pagos y adjuntar comprobantes.

## Funcionalidades

### Búsqueda de Pagos
- Permite buscar pagos por RUT del participante.
- Muestra un listado de pagos registrados con detalles relevantes.

### Registro de Pagos
- Interfaz para registrar un nuevo pago.
- Posibilidad de adjuntar comprobantes de pago.

### Actualización de Dashboard
- Al registrar un pago, se actualiza automáticamente el dashboard financiero.
- Se generan reportes de pagos realizados y pendientes.

## Criterios de Aceptación
- La búsqueda por RUT debe devolver resultados correctos.
- Los pagos registrados deben aparecer en la base de datos.
- El dashboard debe reflejar los cambios en tiempo real.

## Notas
- Asegurarse de que la interfaz sea intuitiva y fácil de usar.
- Implementar validaciones para evitar errores en el registro de pagos.
