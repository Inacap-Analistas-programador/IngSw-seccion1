# Panel Admin: Habilitación / Selección Manual + Export de correos

## Descripción
Este documento detalla la interfaz del panel de administración que permite habilitar usuarios de manera manual, seleccionar múltiples usuarios y exportar listas de correos electrónicos. 

## Funcionalidades
- **Habilitación de Usuarios**: Permite a los administradores marcar usuarios como habilitados para participar en el sistema.
- **Selección Múltiple**: Interfaz que permite seleccionar varios usuarios a la vez para habilitarlos.
- **Exportación de Correos**: Funcionalidad para exportar una lista de correos electrónicos de los usuarios habilitados.

## Criterios de Aceptación
- La interfaz debe permitir la selección de múltiples usuarios.
- La exportación debe incluir las columnas requeridas (nombre, correo, estado).
- La acción de habilitación debe quedar registrada en el sistema.

## Mockups
Incluir enlaces a los mockups de la interfaz de usuario para la habilitación y exportación de correos.

## Consideraciones Técnicas
- Implementar validaciones para evitar habilitar usuarios que ya están habilitados.
- Asegurar que la exportación cumpla con los estándares de formato (CSV, XLSX).

## Notas
Este panel es crucial para la gestión de usuarios y su correcta habilitación en el sistema. Se debe asegurar que la interfaz sea intuitiva y fácil de usar para los administradores.
