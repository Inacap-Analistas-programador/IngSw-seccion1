# Prototipo de Landing Page

## Descripción General
Este documento presenta el prototipo de alta fidelidad para la landing page del sistema de preinscripción. El diseño incluye elementos clave que guiarán al usuario hacia la acción deseada de "Realizar Preinscripción".

## Elementos del Prototipo

### 1. Encabezado
- **Logo**: Ubicado en la esquina superior izquierda.
- **Menú de Navegación**: Enlaces a secciones como "Inicio", "Sobre Nosotros", "Contacto".

### 2. Sección Hero
- **Imagen de Fondo**: Visual atractiva relacionada con el curso.
- **Título Principal**: "¡Inscríbete Ahora!"
- **Subtítulo**: "Aprovecha nuestra oferta educativa."
- **Botón CTA**: "Realizar Preinscripción" (destacado en color).

### 3. Beneficios del Curso
- **Lista de Beneficios**: 
  - Aprendizaje práctico.
  - Asesoría personalizada.
  - Certificación al finalizar.

### 4. Testimonios
- **Citas de Estudiantes**: Comentarios positivos sobre el curso.
- **Imágenes de Estudiantes**: Fotos de participantes anteriores.

### 5. Llamada a la Acción
- **Reiteración del Botón CTA**: "Realizar Preinscripción" al final de la página.

### 6. Pie de Página
- **Información de Contacto**: Dirección, teléfono y correo electrónico.
- **Enlaces a Redes Sociales**: Iconos de Facebook, Twitter, Instagram.
